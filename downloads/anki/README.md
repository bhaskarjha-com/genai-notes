# Anki-Friendly Interview Decks

> These exports are tab-separated text decks that import directly into Anki without any extra packaging toolchain.

## Import Steps

1. Open Anki and choose File -> Import.
2. Select one of the .tsv files in this folder.
3. Map column 1 to the front field and column 2 to the back field.
4. Use the filename as the deck name, or import into a custom deck.
